My Career

frontend Developer

Every website has a frontend; it’s everything about the service a user will actually see. Because it’s what users interact with, the frontend has to be accessible, well-designed, intuitive for the user, and it has to perform well. Frontend development is a specialism of software development. In simple terms, the frontend of a website is the part users actually see and interact with, what’s known as the user interface (UI). Every website has a UI in some form.
